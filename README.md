# Документация проекта
## Описание проекта

Автотесты для интернет-магазина SauceDemo, проверяют функционал оформления заказа на сайте https://www.saucedemo.com/. 
Использованы:
- Page Object Model,
- Firefox,
- Allure.

## Форматирование кода
Использованы: Python с библиотеками **Selenium, Pytest и Allure**. Код сответсвует стандарту **PEP8**. Методы классов указаны с возвращаемыми значений.

## Инструкция по запуску тестов
* Установите зависимости:
bash
pip install -r requirements.txt

* Запустите тесты с генерацией отчета Allure:
bash
pytest --alluredir=./allure-results

### Инструкция по просмотру отчета Allure

* После выполнения тестов запустите:
bash
allure serve ./allure-results

### Отчет содержит:

- Название теста: "Процесс оформления заказа с проверкой суммы"

- Описание теста: Тестирование оформления заказа с тремя товарами

**Шаги теста:**

1.Открытие сайта и авторизация

2.Добавление товаров в корзину

3.Переход в корзину

4.Начало оформления заказа

5.Заполнение данных

6.Проверка итоговой суммы

**Успешное выполнение считается за выведение суммы $58.29**

## Используемые методы классов (кроме self)
* Метод: open()
Возвращаемое значение: None
Описание: Открывает страницу авторизации по URL https://www.saucedemo.com/

* Метод: enter_username(username: str)
Параметры:username: str - имя пользователя для ввода
Возвращаемое значение: None
Описание: Вводит имя пользователя в поле ввода

* Метод: enter_password(password: str)
Параметры: password: str - пароль для ввода
Возвращаемое значение: None
Описание: Вводит пароль в поле ввода

* Метод: click_login()
Возвращаемое значение: None
Описание: Нажимает кнопку входа

* Метод: login(username: str, password: str)
Параметры:username: str - имя пользователя
password: str - пароль
Возвращаемое значение: None
Описание: Выполняет полный процесс авторизации

* Метод: get_error_message()
Возвращаемое значение: str
Описание: Возвращает текст сообщения об ошибке авторизации

* Метод: is_loaded()
Возвращаемое значение: bool
Описание: Проверяет, загружена ли страница авторизации

* Класс: ProductsPage (страница товаров)
Метод: is_loaded()
Возвращаемое значение: bool
Описание: Проверяет, загружена ли страница товаров

*Метод: add_backpack_to_cart()
Возвращаемое значение: None
Описание: Добавляет товар "Sauce Labs Backpack" в корзину

* Метод: add_bolt_t_shirt_to_cart()
Возвращаемое значение: None
Описание: Добавляет товар "Sauce Labs Bolt T-Shirt" в корзину

* Метод: add_onesie_to_cart()
Возвращаемое значение: None
Описание: Добавляет товар "Sauce Labs Onesie" в корзину

* Метод: add_all_required_items()
Возвращаемое значение: None
Описание: Добавляет все три требуемых товара в корзину

* Метод: get_cart_items_count()
Возвращаемое значение: int
Описание: Возвращает количество товаров в корзине

* Метод: go_to_cart()
Возвращаемое значение: None
Описание: Переходит на страницу корзины

Класс: CartPage (страница корзины)
* Метод: is_loaded()
Возвращаемое значение: bool
Описание: Проверяет, загружена ли страница корзины

* Метод: get_cart_items_count()
Возвращаемое значение: int
Описание: Возвращает количество товаров в корзине

* Метод: get_cart_item_names()
Возвращаемое значение: list[str]
Описание: Возвращает список названий товаров в корзине

* Метод: click_checkout()
Возвращаемое значение: None
Описание: Нажимает кнопку "Checkout" для начала оформления заказа

* Метод: click_continue_shopping()
Возвращаемое значение: None
Описание: Нажимает кнопку "Continue Shopping" для возврата к товарам

* Класс: CheckoutPage (страницы оформления заказа)
Метод: is_checkout_info_loaded()
Возвращаемое значение: bool
Описание: Проверяет, загружена ли страница информации для оформления

* Метод: is_checkout_overview_loaded()
Возвращаемое значение: bool
Описание: Проверяет, загружена ли страница обзора заказа

* Метод: fill_checkout_info(first_name: str, last_name: str, zip_code: str)
Параметры:
first_name: str - имя покупателя
last_name: str - фамилия покупателя
zip_code: str - почтовый индекс
Возвращаемое значение: None
Описание: Заполняет форму информации для оформления заказа

* Метод: click_continue()
Возвращаемое значение: None
Описание: Нажимает кнопку "Continue" для перехода к обзору заказа

* Метод: click_finish()
Возвращаемое значение: None
Описание: Нажимает кнопку "Finish" для завершения заказа

* Метод: is_order_complete()
Возвращаемое значение: bool
Описание: Проверяет, завершен ли заказ

* Метод: get_complete_message()
Возвращаемое значение: str
Описание: Возвращает сообщение о завершении заказа
