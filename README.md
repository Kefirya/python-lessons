# Документация проекта
## Описание проекта

Автотесты для интернет-магазина SauceDemo, проверяют функционал оформления заказа на сайте https://www.saucedemo.com/. 
Использованы:
- Page Object Model,
- Firefox,
- Allure.

## Форматирование кода
Использованы: Python с библиотеками **Selenium, Pytest и Allure**. Код сответсвует стандарту **PEP8**. Методы классов указаны с возвращаемыми значений.

## Инструкция по запуску тестов
* Установите зависимости:
bash
pip install -r requirements.txt

* Запустите тесты с генерацией отчета Allure:
bash
pytest --alluredir=./allure-results

### Инструкция по просмотру отчета Allure

* После выполнения тестов запустите:
bash
allure serve ./allure-results

### Отчет содержит:

- Название теста: "Процесс оформления заказа с проверкой суммы"

- Описание теста: Тестирование оформления заказа с тремя товарами

**Шаги теста:**

1.Открытие сайта и авторизация

2.Добавление товаров в корзину

3.Переход в корзину

4.Начало оформления заказа

5.Заполнение данных

6.Проверка итоговой суммы

**Успешное выполнение считается за выведение суммы $58.29**
